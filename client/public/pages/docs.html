Task Manager Web App Documentation

Overview

The Task Manager Web App is a powerful tool designed to help users manage their tasks efficiently. It allows individuals and teams to create, organize, and track tasks, set deadlines, and collaborate seamlessly. The application is built using modern web technologies and follows best practices for performance and usability.

Features

  • User Authentication: Secure login and registration for users.
  • Task Management: Create, edit, delete, and prioritize tasks.
  • Due Dates and Reminders: Set deadlines and receive notifications.
  • Collaboration: Share tasks with team members and assign responsibilities.
  • Search and Filter: Easily find tasks using search and filter options.
  • Responsive Design: Access the app on any device, including mobile and desktop.

Installation

Prerequisites

  • Node.js (version 14 or higher)
  • npm (Node Package Manager)
  • MongoDB

Steps

  1. Clone the Repository: git clone https://github.com/your-username/task-manager.git cd task-manager
  2. Install Dependencies: 
    npm install
  3. Set Up Environment Variables:

    Create a .env file in the root directory and add the following variables: 

    PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
  4. Run the Application: 
    Open index.html in your browser

Usage

Once the application is running, you can:

  1. Register a New Account: Click on the "Sign Up" button and fill in the required information.
  2. Log In: Use your credentials to log in to the application.
  3. Create a Task: Click on the "Add Task" button, fill in the task details, and save.
  4. Manage Tasks: Edit, delete, or mark tasks as complete from your task list.

Authentication

All API requests require authentication via a JSON Web Token (JWT). To obtain a token, users must log in with their credentials.

Login Endpoint

POST /api/auth/login

Request Body:

{
    "email": "user@example.com",
    "password": "yourpassword"
}

Response:

{
    "token": "your_jwt_token"
}
